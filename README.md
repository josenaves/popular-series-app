# Popular Series

A simple app to discover popular shows in a grid layout.

## Features
- Discover the most popular shows
- Detail shows' screen
- Save favorite shows locally

## API Reference

The app uses Trakt.tv API to consume popular shows' information.

## Libraries

- Retrofit
- Picasso
